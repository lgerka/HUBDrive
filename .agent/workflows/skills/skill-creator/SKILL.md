# Skill Creator

This skill provides guidance for creating effective skills.

## Quick Start Guide

To access specific parts of the Skill Creator guide, please refer to the following resources:

1.  **Understanding Skills**: For an overview of what skills are and what they provide, see [references/about.md](references/about.md).
2.  **Core Principles**: For principles like "Concise is Key" and "Progressive Disclosure", see [references/principles.md](references/principles.md).
3.  **Skill Structure**: For detailed anatomy of a skill and file organization, see [references/structure.md](references/structure.md).
4.  **Creation Process**: For the step-by-step guide to creating, initializing, and packaging a skill, see [references/process.md](references/process.md).

## Usage

When creating a new skill, follow the **Creation Process** outlined in [references/process.md](references/process.md). Use `scripts/init_skill.py` to start a new project and `scripts/package_skill.py` to finalize it.
